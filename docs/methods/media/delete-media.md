Delete media file using Media Id

```dart
await whatsapp.deleteMedia(
  mediaId: 'MEDIA_ID',
);
```

| param     | type   | required | description               |
| --------- | ------ | -------- | ------------------------- |
| `mediaId` | String | yes      | Media Id of uploaded file |
